# MediaX
MediaUtil For Android

使用  
1.Add it in your root build.gradle at the end of repositories:
```gradle
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
2.Add the dependency
```gradle
	dependencies {
	        implementation 'com.github.wenvelope:MediaX:1.2'
	}
```
